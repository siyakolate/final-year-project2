# 🌿 Lakshmi Ayurvedic – Frontend

This is the frontend of the Lakshmi Ayurvedic Herbal Store Website, developed using React JS. It provides a user-friendly interface for customers to explore Ayurvedic products, send enquiries, and submit feedback.

---

## 🚀 Features

- 🧾 View Ayurvedic medicines and herbal products  
- 📩 Enquiry form for customer interaction  
- ⭐ Feedback submission system  
- 📱 Responsive design for mobile and desktop  
- 🔐 Admin interface (UI)  

---

## 🛠️ Tech Stack

- React JS  
- HTML5  
- CSS3  
- JavaScript
